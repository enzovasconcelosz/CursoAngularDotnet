Detalhes do perfil

O campo "Título" é obrigatório.
O campo "Nome" é obrigatório.
O campo "Nome" deve conter no mínimo 2 caracteres.
O campo "Nome" deve conter no máximo 15 caracteres.
O campo "Sobrenome" é obrigatório.
O campo "Sobrenome" deve conter no mínimo 2 caracteres.
O campo "Sobrenome" deve conter no máximo 50 caracteres.
O campo "E-mail" é obrigatório.
O e-mail digitado não é válido.
O campo "E-mail" deve conter no máximo 40 caracteres.
O campo "Telefone" é obrigatório.
O campo "Função" é obrigatório.
O campo "Descrição" é obrigatório.

Alteração de senha

Caso necessite alterar a senha, preencha os campos abaixo:

O campo "Senha" é obrigatório.
O campo "Senha" deve conter no mínimo 6 caracteres.
O campo "Senha" deve conter no máximo 20 caracteres.
O campo "Confirmar senha" é obrigatório.
O campo "Confirmar senha" deve conter no mínimo 6 caracteres.
O campo "Confirmar senha" deve conter no máximo 20 caracteres.
A senha informada não está condizente com o campo "Senha".
